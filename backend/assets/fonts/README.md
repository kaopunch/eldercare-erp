# PDF Thai Fonts

ElderCare ERP bundles Noto Sans Thai for invoice and receipt PDF rendering.

Files:
- `NotoSansThai-Regular.ttf`
- `NotoSansThai-Bold.ttf`
- `OFL.txt`

Source: Google Noto Fonts project.
License: SIL Open Font License 1.1.

The backend also supports overriding these files through:
- `ELDERCARE_PDF_FONT_PATH`
- `ELDERCARE_PDF_BOLD_FONT_PATH`
- `ELDERCARE_PDF_FONT_FACE`
- `ELDERCARE_PDF_BOLD_FONT_FACE`
