# Training Manual

## Module 1: Basic Elderly Care
- การสื่อสารกับผู้สูงวัย
- การพยุงที่ปลอดภัย
- การช่วยขึ้นลงรถ
- ข้อห้าม: ห้ามยกคนเดียวหากเสี่ยงบาดเจ็บ

## Module 2: Safety Driving
- ขับนุ่มนวล
- ระยะเบรก
- เข้าโค้งช้า
- หลีกเลี่ยงถนนกระแทกถ้าเป็นไปได้

## Module 3: Service Mindset
- พูดช้า ชัด สุภาพ
- ไม่โต้เถียง
- ไม่แสดงอารมณ์หงุดหงิด
- ให้เกียรติผู้สูงวัย

## Module 4: ERP SOP
- รับงาน
- เช็คอิน
- อัปเดตสถานะ
- ส่งรูป
- ปิดงาน

## Module 5: Incident Response
- ประเมินความรุนแรง
- โทร dispatcher
- บันทึกเหตุ
- ห้ามปกปิดเหตุการณ์
