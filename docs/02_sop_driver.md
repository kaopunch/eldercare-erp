# SOP คนขับและผู้ช่วยดูแล

## ก่อนรับงาน
1. ตรวจรายละเอียด booking
2. ตรวจชื่อผู้สูงวัย จุดรับ จุดส่ง เวลา และข้อควรระวัง
3. โทรยืนยันกับผู้จอง
4. ตรวจรถ ความสะอาด น้ำมัน อุปกรณ์ช่วยเหลือพื้นฐาน

## ถึงหน้างาน
1. กด arrived ในระบบ
2. ยืนยันตัวตนผู้สูงวัย
3. ประเมินสภาพเบื้องต้น เช่น เดินไหวไหม เวียนหัวไหม
4. ช่วยขึ้นรถอย่างช้าและปลอดภัย
5. กด elder_onboard

## ระหว่างเดินทาง
1. ขับนุ่มนวล ไม่เร่ง ไม่เบรกแรง
2. ห้ามรับงานซ้อน
3. ห้ามเปิดเผยข้อมูลส่วนตัวผู้สูงวัย
4. หากมีอาการผิดปกติ โทร dispatcher ทันที

## ส่งปลายทาง
1. ส่งถึงบุคคล/สถานที่ที่กำหนด
2. ถ่ายรูปหลักฐานตาม policy
3. กด handover_completed
4. บันทึก note ถ้ามีเหตุการณ์พิเศษ

## กรณี Incident
- Low: บันทึกในระบบ
- Medium: แจ้ง dispatcher
- High/Critical: หยุดงาน แจ้ง admin และติดต่อญาติทันที
